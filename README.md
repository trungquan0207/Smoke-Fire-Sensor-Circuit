# Smoke-Fire-Sensor-Circuit
Smoke and Fire Detector Circuit (First PCB Project)
🚀 Learning Objectives
As a starter project, I focused on:

Understanding the basic working principles of MQ-series smoke sensors and Infrared (IR) flame sensors.

Learning the end-to-end PCB design workflow: from Schematic capture to PCB Layout.

Getting familiar with electronic component footprints and soldering techniques.

🛠 Features
Dual Detection: Detects both smoke particles and infrared light from flames.

Alert System: Integrated Buzzer/LED indicators for immediate local alarm.

Adjustable Sensitivity: Uses onboard potentiometers to tune detection thresholds.
